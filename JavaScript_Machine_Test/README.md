# TO DO APP
###### You Can Add Your To Do List, By Writing In The Input Tab, And After It You Can Click Add Button To Add the List, Also You Can Clear all Your to Do By Clicking On The Clear all Button  

# Stacks Used
## Nodejs
## Expressjs
## Templete Engines
## Bootstrap For Frontend
## CSS3 Only For UI Background

# Requirements
## Node 8
## Git
